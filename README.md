## Projeyi Kendi Bilgisayarınızda Çalıştırmak İçin:

1. ```git clone https://github.com/berkbal/docs.git```
2. ```cd docs```
3. ```python3 -m venv venv```
4. ```source venv/bin/activate```
5. ```pip install -r requirements.txt```
6. ```mkdocs serve```
